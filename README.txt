# AJCZ Web Agency - Installer Package

## 📦 Contenido del Paquete

Este paquete contiene todo lo necesario para instalar AJCZ Web Agency en cualquier computadora con Windows.

### 📁 Archivos Incluidos:

- **`AJCZ-Web-Agency.exe`** - Aplicación principal (194 MB)
- **`install.bat`** - Instalador automático
- **`icon.ico`** - Icono de la aplicación
- **`README.txt`** - Este archivo de instrucciones

## 🚀 Cómo Instalar

### Método 1: Automático (Recomendado)
1. **Ejecutar como administrador**: Clic derecho en `install.bat` → "Ejecutar como administrador"
2. **Confirmar instalación**: Escribir `Y` cuando pregunte
3. **Listo**: La app se instalará en `C:\Program Files\AJCZ Web Agency\`

### Método 2: Manual
1. Crear carpeta: `C:\Program Files\AJCZ Web Agency\`
2. Copiar `AJCZ-Web-Agency.exe` a esa carpeta
3. Crear acceso directo en el escritorio

## ✅ Requisitos del Sistema

- **Windows 10 o 11** (64-bit)
- **4 GB RAM** mínimo
- **200 MB** de espacio en disco
- **Conexión a internet** (para el contenido web)

## 🎯 Características de la Aplicación

- ✅ Landing page animada con diseño cyberpunk
- ✅ Integración con WhatsApp
- ✅ Interfaz PyQt6 moderna
- ✅ Sin dependencias adicionales
- ✅ Portable (puede ejecutarse desde USB)

## 📱 Acceso Directo Creado

Tras la instalación, encontrarás:
- **Escritorio**: "AJCZ Web Agency.url"
- **Menú Inicio**: "AJCZ Web Agency"

## 🔧 Soporte

Si tienes problemas:
1. Ejecuta `install.bat` como administrador
2. Verifica que Windows Defender no bloquee archivos
3. Asegúrate de tener 200 MB libres en C:\
