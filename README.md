## 🚀 Installation & Usage

### For Desktop (Windows)
1. **Run as Administrator**: Right-click `install.bat` → "Run as administrator"
2. **Confirm Installation**: Type `Y` when prompted
3. **Done**: App installs to `C:\Program Files\AJCZ Web Agency\`

### For Mobile/Web
1. **Open in Browser**: Open `ajcz_animacion.html` in any web browser
2. **Works on All Devices**: Mobile, tablet, desktop
3. **No Installation Required**: Instant access
